# Installing

```
npm install @hellocoop/svelte-otp
```

or

```
yarn add @hellocoop/svelte-otp
```

<br/>
<br/>

# Usage

## Simplest form

Getting started is as easy as importing the `SvelteOtp` component and just using it.

<br/>

```js
<script>
    import SvelteOtp from ‘@hellocoop/svelte-otp’;
</script>

<SvelteOtp />
```
